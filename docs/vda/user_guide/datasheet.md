## 激光测距仪

激光测距仪位于图像采集模块的鹰眼盒中，相机镜头下方 3cm 处，使用时请勿直视或遮挡。激光测距仪参数如下表。

| 参数 | 数值 |
| ---- | ---- |
| 精度 | ±1 mm |
| 量程 | 0.1—40 m |
| 激光类型 | 630—670 nm，ClassII，<1 mW |
